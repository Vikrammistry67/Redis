# Node.js + Redis Projects

This repo contains backend projects using Node.js and Redis.

## Projects

- OTP Verification System (with TTL)
- User API (check user exists in Redis)
- Banner API (basic backend logic)

## Tech Stack

- Node.js
- Express.js
- Redis

## Redis Used For

- OTP storage with expiry
- User caching
- Fast data access

## Redis Commands Used

- SET, GET, DEL
- HSET, HGET, HEXISTS
- EXPIRE

## Run Project

npm install  
npm start
