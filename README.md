# Node.js + Redis Projects

This repo contains backend projects using Node.js and Redis.

## Projects

* OTP Verification System (with TTL)
* User API (check user exists in Redis)
* Banner API (basic backend logic)
* Email Queue System using Redis Lists

## Tech Stack

* Node.js
* Express.js
* Redis
* Morgan 
* dotenv

## Redis Used For

* OTP storage with expiry
* User caching
* Fast data access
* Email queue management

## Redis Commands Used

### String Commands

* SET
* GET
* DEL
* EXPIRE

### Hash Commands

* HSET
* HGET
* HEXISTS

### List Commands

* LPUSH
* RPUSH
* LPOP
* RPOP
* LRANGE

## Features

* OTP Verification with TTL
* Redis Hash Operations
* Redis List Operations
* Email Queue Processing
* User Existence Check
* Fast Data Retrieval

npm install

npm start
